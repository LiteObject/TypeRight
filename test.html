Debugging Checklist:

  1. Is Ollama running? Run: ollama serve
  2. Open Chrome DevTools (F12) → Console tab
  3. Look for "TypeRight: Content script initialized" message
  4. Type in a field and look for "TypeRight: Checking grammar" messages
  5. Check for any error messages in red
  6. Go to chrome://extensions/ → TypeRight → "Inspect views: service worker"
  7. Check background script console for errors
